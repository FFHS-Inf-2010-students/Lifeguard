Eclipse Project Setup for Android with EGit
===========================================

1. Install EGit via Eclipse Marketplace
2. In Package Explorer: right-click > Import...
3. Git > Projects from Git. Next
4. URI: https://github.com/FFHS-Inf-2010-students/Lifeguard.git
   User: ...
   Password: ...
   Store in Secure Store: [x]. Next. Next.
5. Select destination... Next.
6. Use the New Project Wizard. Finish.
7. Android > Android Project from Existing Code. Next.
8. Root Directory: Selected Directory of Step 5.
   Change New Project Name to Lifeguard. Finish.
